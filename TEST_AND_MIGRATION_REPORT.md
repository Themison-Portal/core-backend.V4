## 3. Deployment Hardening Details

### 🟢 CORS Policy Improvement
*   **Issue**: Relative URLs or trailing slashes in `FRONTEND_URL` were causing pre-flight (OPTIONS) failures.
*   **Fix**: Implemented automatic string sanitization to strip trailing slashes, ensuring robust cross-origin resource sharing.

### 🟢 Timezone-Aware Authentication
*   **Issue**: `datetime.utcnow()` deprecation and local server offsets were causing premature session/token expiration.
*   **Fix**: Migrated to `datetime.now(timezone.utc)` across the authentication flow for absolute consistency.
